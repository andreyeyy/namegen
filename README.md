# namegen
Namegen is a command line tool to generate any number of random names.
## Usage
1. Either compile namegen.c using ```gcc namegen.c -o namegen ```, or get a compiled binary from releases.
2. Run:

```./namegen``` to generate one random name 

```./namegen 55``` to generate 55 random names.
